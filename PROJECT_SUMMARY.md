## 🛠️ 技术栈详情

### 前端框架
- **Next.js 14**: 使用 App Router，支持 SSG/ISR
- **React 19**: 最新版本的 React
- **TypeScript**: 类型安全的 JavaScript

### 样式方案
- **Tailwind CSS 4**: 实用优先的 CSS 框架
- **shadcn/ui**: 现代化组件库
- **PostCSS**: CSS 后处理器

### 开发工具
- **ESLint**: 代码质量检查
- **Turbopack**: 快速构建工具
- **Hot Reload**: 热重载开发体验
